# @llmletsplay/versus-omok Rules

These notes describe the gameplay currently implemented by this package so downstream apps know exactly what the engine expects.

## Objective

Place five of your stones in an unbroken line before your opponent does.

## Players

2 players.

## Setup

- The game starts on an empty board with the first player to move.

## Turn Structure

- Place one stone on any empty legal point.

## End Of Game

- The first player to create five consecutive stones horizontally, vertically, or diagonally wins.
