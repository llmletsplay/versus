# @versus/tic-tac-toe Rules

These notes describe the gameplay currently implemented by this package so downstream apps know exactly what the engine expects.

## Objective

Make three in a row before your opponent does.

## Players

2 players.

## Setup

- The board starts empty with X to move first.

## Turn Structure

- Place your mark in any empty cell.

## End Of Game

- Three in a row wins and a full board without a line is a draw.
